Overdrive

Harness Drive's real-time features to collaborate in Sublime Text

Install

Simply clone the repo into your Sublime Text Packages folder:

hg clone \
https://bitbucket.org/marksteve/overdrive \
$ST_PACKAGES_DIR/Overdrive
